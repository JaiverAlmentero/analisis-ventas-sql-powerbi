# Proyecto: Análisis de Ventas

Este repositorio contiene un proyecto completo de análisis de ventas realizado con **SQL Server** y **Power BI**.

## Archivos

- `proyecto_venta.SQL` → Consultas SQL para analizar los datos de ventas. Comentadas y organizadas.
- `DATASET.CSV.pbix` → Dataset de ventas (puede abrirse en Power BI para interactuar con los datos).
- `Proyec_venta_Bi.pbix` → Dashboard de Power BI con 3 pestañas de análisis:
  1. Ventas por ciudad
  2. Productos más vendidos
  3. Porcentaje de ventas por ciudad

## Objetivo

Analizar los datos de ventas para identificar:
- Ingresos por ciudad
- Productos más vendidos
- Tendencias de ventas
- Preparar dashboards interactivos en Power BI

## Herramientas utilizadas

- SQL Server
- Power BI

## Notas

- El archivo SQL está listo para ejecutar en SQL Server.
- El dashboard de Power BI contiene todas las pestañas en un solo archivo `.pbix`.
